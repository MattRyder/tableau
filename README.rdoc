= Tableau

Parses, analyses and creates Timetables for Staffordshire University students.

== Installing Tableau

To install Tableau to your project, include it in your `Gemfile`.

== Developing Tableau

To develop the gem further, fork and clone to your local machine, then run:

` mkdir /spec/dummy/db; rake db:create `

to create the databases for RSpec.

Ensure your changes are working and accompanied by a test suite before issuing a pull request.
